   - Alt text for images
   - Submit sitemap

### Phase 2: Social Media Setup (Week 2-3)
1. **Design carousel graphics**
   - Use templates in `social-media-carousels.md`
   - Create in Canva or Photoshop
   - Follow brand guidelines
   - Export as 1080x1080px

2. **Film video reels**
   - Use scripts in `video-reel-scripts.md`
   - Professional lighting and audio
   - Add captions (required)
   - Edit with CapCut or Premiere

3. **Set up ManyChat**
   - Follow `manychat-automation-guide.md`
   - Configure all 18 trigger words
   - Test each automation
   - Create resource PDFs

   - Use `3-month-content-calendar.md`
   - Schedule via Later, Buffer, or native tools
   - Prepare first month in advance

### Phase 3: Digital Products (Week 4-6)
1. **Create lead magnets (Priority 1)**
   - SEEDS Method Overview Guide
   - Bridge Model Template
   - 5x5 Prospecting Planner
   - Trust Builders Guide

2. **Write e-books**
   - Use outlines in `ebook-outlines.md`
   - Design in Canva or InDesign
   - Create downloadable PDFs
   - Set up delivery system

3. **Build DIY courses**

   - Create workbooks
   - Set up on Teachable or Kajabi
   - Price at $97-$397

### Phase 4: Launch & Optimize (Week 7-12)
1. **Soft launch**
   - Announce to email list
   - Post on social media
   - Offer early bird pricing

2. **Monitor and adjust**
   - Track website analytics
   - Review social media engagement
   - Optimize ManyChat sequences
   - A/B test messaging

3. **Scale**
   - Increase posting frequency
   - Run paid ads
   - Partner with influencers
   - Launch affiliate program
