# ManyChat Automation Guide for CGN
## Instagram DM Automation Strategy

**Purpose:** Convert social media engagement into qualified leads through automated, personalized responses
**Platforms:** Instagram DMs (primary), Facebook Messenger (secondary)
**Goal:** Nurture leads from comment to discovery call booking

---

## ManyChat Setup Overview

### Step 1: Connect Accounts
- Link Instagram Business Account to ManyChat
- Connect Facebook Page to ManyChat
- Verify admin permissions
- Test connection with sample trigger

### Step 2: Configure Automation Settings
- Enable auto-response for comments
- Set response timing (instant recommended)
- Configure working hours (or 24/7)
- Set up notification alerts for human takeover

### Step 3: Create Trigger Keywords
- Set up case-insensitive matching
- Allow for common misspellings
- Configure trigger conditions (must contain keyword)
- Set up cooldown periods (prevent spam)

## Follow-Up Sequence Strategy

### General Follow-Up Timeline

**Immediate (0-5 minutes):**
- Automated trigger response delivered

**Day 1 (If they respond):**
- Automated qualifying question follow-up
- OR human takeover if complex question

**Day 2 (If no response):**
- Automated check-in message
- "Did you get a chance to check out [resource]?"

**Day 4 (If still no response):**
- Automated value-add message
- Share related content or additional resource

**Day 7 (If still no response):**
- Automated final touchpoint
- "Last call" or "Still interested?" message
- Offer alternative ways to engage

**After Day 7:**
- Add to email nurture sequence
- No more automated DMs (avoid being spammy)
- Human can re-engage manually later

## Human Takeover Triggers

**When to have a human step in:**

1. **They ask a specific question** not covered by automation
2. **They express serious buying intent** ("How much is...?", "When can we start?")
3. **They mention a problem you can solve** specifically
4. **They've engaged 3+ times** with automated messages
5. **They've booked a call** (send personal confirmation)
6. **They mention competitors** or comparison shopping
7. **They express frustration or confusion**
8. **High-value prospect** (based on their profile)

**Notification Settings:**
- Alert team member immediately for high-priority takeovers
- Daily digest for moderate priority
- Track response time (goal: < 2 hours during business hours)

## ManyChat Setup Best Practices

### 1. Keyword Configuration
- **Case Insensitive:** Yes
- **Match Type:** Contains keyword (allows for "SEEDS please" to work)
- **Duplicate Prevention:** 24-hour cooldown per user per keyword
- **Testing:** Test each trigger before going live

### 2. Message Templates
- **Personalization:** Use {First Name} token
- **Emojis:** 2-3 per message (but not excessive)
- **Link Format:** Use shortened, trackable links
- **Character Limit:** Keep messages under 1000 characters
- **Button Options:** Use buttons for multi-choice responses

### 3. Compliance
- **Permission-Based:** Only DM people who commented
- **Opt-Out:** Include "Reply STOP to unsubscribe" in first message
- **Privacy:** Don't share or sell DM lists
- **Instagram Rules:** Follow Instagram's automation policies

### 4. Tracking & Analytics
- **Monitor Metrics:**
  - Trigger usage (which keywords are most popular)
  - Open rates (did they see the DM?)
  - Click rates (did they click the link?)
  - Response rates (did they engage back?)
  - Conversion rates (did they book a call or buy?)

- **Weekly Review:**
  - Top performing triggers
  - Drop-off points in sequences
  - Quality of leads generated
  - Adjustments needed

### 5. A/B Testing
**Test Variables:**
- Message length (short vs. detailed)
- Tone (formal vs. casual)
- Number of questions asked
- CTA placement
- Emoji usage
- Link vs. button

**Testing Process:**
- Run variation for 2 weeks
- Minimum 50 uses per variation
- Analyze conversion to next step
- Implement winner

## Troubleshooting Common Issues

**Issue:** People comment but don't see DM
**Solution:** Make sure they follow your account first (mention in post)

**Issue:** High drop-off after first message
**Solution:** First message may be too long or not valuable enough

**Issue:** Low click-through on resources
**Solution:** Link might look spammy; use recognizable domain

**Issue:** Too many unqualified leads
**Solution:** Add qualifying question earlier in sequence

**Issue:** Human takeover happening too late
**Solution:** Adjust notification triggers to be more aggressive
